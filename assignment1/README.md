# Assignment 1

JavaScript fundamentals assignment covering coding exercises, a bonus closures problem, and conceptual essay questions.

## Contents

- [solution.js](solution.js) — Part 1: 15 coding questions covering type conversion, truthy/falsy checks, loops, array methods (`filter`, `map`, `reduce`), the spread operator, `switch` statements, arrow functions, destructuring, rest parameters, and promises.
- [bonus.js](bonus.js) — Bonus question: a `createCounter` closure with `increment`, `decrement`, and `reset` methods.
- [essay.md](essay.md) — Part 2: 5 essay questions on `forEach` vs `for...of`, hoisting and the Temporal Dead Zone, `==` vs `===`, `try-catch` in async operations, and type conversion vs. coercion.

## Running the code

Each JS file can be run directly with Node:

```bash
node solution.js
node bonus.js
```
